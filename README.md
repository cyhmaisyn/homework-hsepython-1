# Домашнее задание по Python

Этот репозиторий создан с целью загрузки первого домашнего задания (и, видимо, всех последующих) в рамках курса по изучению Python на 4 курсе.

## Содержание репозитория

Репозиторий включает в себя следующие файлы / папки:

- **HW** - папка с домашними заданиями:
  - `Малыгина_БСЦ_223_HW1.ipynb` - файл с решением первого домашнего задания
- **requirements.txt** - список библиотек Python или же файл, где фиксируются библиотеки из виртуального окружения;
- **LICENSE** - файл лицензии MIT (разрешение на использование кода);
- **.gitignore** - файл, куда подгружено виртуальное окружение.
